# 💻 Smart Laptop Finder

An intelligent laptop recommendation platform that helps users find the most suitable laptop based on their budget, usage, preferred brand, and priorities. The platform also compares prices across multiple online stores, inspired by BuyHatke, to help users make informed purchasing decisions.

---

## 🚀 Features

- 🎯 Smart laptop recommendation engine
- 💰 Budget-based filtering
- 💼 Usage-based recommendations (Coding, Gaming, Business, Student, Creator)
- 🏷️ Brand preference selection
- 🔍 Natural language search
- 🛒 Multi-store price comparison
- 🔄 BuyHatke-inspired price sync feature
- 📱 Fully responsive UI
- ⚡ Fast and interactive user experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🧠 How It Works

1. User enters their requirements.
2. The recommendation engine evaluates laptops using a weighted scoring algorithm.
3. The best matching laptops are displayed.
4. Users can compare prices across different stores.
5. Users choose the best deal available.
